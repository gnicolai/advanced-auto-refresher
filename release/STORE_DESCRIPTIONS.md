## English

### Short Description
Auto-refresh tabs with smart timing, numeric and text monitoring, visual highlights, and Telegram alerts.

### What's New in v1.0.21
- Added text and keyword monitoring alongside numeric monitoring
- Added full page text and full page HTML monitoring
- Added visual highlights and clickable in-page change toasts
- Added optional stop refresh on page click
- Added browser and Telegram alerts for both numeric and text changes
- Improved timer stability for Manifest V3 service worker lifecycle
- Added separate or shared alert sounds for numeric and text monitoring

### Detailed Description
**Advanced Auto Refresher** is a powerful Chrome extension built for users who need more than a simple page reload button.

Use it to auto-refresh tabs with smart stochastic timing and monitor pages for **numeric changes**, **text changes**, **keywords**, or **HTML changes**.

#### Smart Auto-Refresh
- Set a custom refresh interval for each tab
- Use stochastic timing to randomize refreshes and reduce predictable patterns
- Choose Uniform or Gaussian distribution
- Improved countdown recovery for Manifest V3

#### Numeric Monitoring
- Watch a selected page element as a number
- Get alerts when the value increases, decreases, changes, or crosses a threshold
- Great for stock checks, counters, availability, and drop monitoring

#### Text and Keyword Monitoring
- Monitor selected element text, full page text, or full page HTML
- Track appearance, disappearance, or text changes
- Add one or more keywords or phrases, one per line
- Use text monitoring together with numeric monitoring on the same tab

#### Alerts and Visual Feedback
- 6 selectable alert sounds per tab: Siren, Beep, Chime, Alarm, Bell, Digital
- Adjustable volume per tab
- Preview button before saving the sound
- Shared or separate alerts for numeric and text monitoring
- Visual highlight when a change is detected
- Clickable in-page toast to jump directly to the changed area
- Optional debug mode to inspect captured text or HTML

#### Telegram Notifications
- Receive Telegram alerts for numeric changes
- Receive Telegram alerts for text changes and keyword matches
- Combined alerts when numeric and text triggers happen together
- Uses your own Telegram bot and chat ID
- No external backend required

#### More Control
- Optional "Stop refresh when user clicks on the page"
- URL blacklist and whitelist with wildcard support
- Multi-language UI with 9 supported languages

Perfect for:
- Amazon Vine users
- E-commerce deal hunters
- Ticket and booking watchers
- Availability and stock tracking
- Power users who need fast page change alerts
