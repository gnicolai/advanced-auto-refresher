### Manual Installation (Developer Mode)
1. Download or clone this repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable **Developer mode** (toggle in top right)
4. Click **Load unpacked** and select the extension folder
5. The extension icon will appear in your toolbar

## ğŸš€ Usage

1. **Click the extension icon** to open the popup
2. **Set your refresh interval** (in seconds)
3. **Enable Stochastic mode** for random timing variation
4. **Click Start** to begin auto-refreshing
5. **Optional**: Enable content monitoring to get alerts

### Content Change Detection
1. Enable **"Monitor content"**
2. Click **"Click to select element"**
3. Click on any element on the page you want to monitor
4. The extension will alert you when that element's value changes
