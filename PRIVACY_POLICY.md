# Privacy Policy for Advanced Auto Refresher

**Last Updated: February 4, 2026**

## Overview

Advanced Auto Refresher ("the Extension") is a browser extension that provides automatic page refresh functionality with content change detection. This privacy policy explains how we handle your data.

## Data Collection

### What We Collect
The Extension does **NOT** collect, store, or transmit any personal data to external servers.

### Local Storage Only
All data is stored locally on your device using Chrome's built-in storage APIs:
- **Tab refresh settings** (intervals, stochastic options)
- **Content watch selectors** (CSS selectors you choose to monitor)
- **Blacklist/Whitelist URLs** (patterns you configure)
- **Language preference** (your chosen interface language)

## Data Usage

Your data is used solely for:
1. Remembering your refresh preferences for each tab
2. Restoring settings when you reopen Chrome
3. Displaying the interface in your preferred language

## Data Sharing

We do **NOT**:
- Share any data with third parties
- Send any data to external servers
- Track your browsing activity
- Collect analytics or usage statistics

## Permissions Explained

The Extension requires certain permissions to function:

| Permission | Purpose |
|------------|---------|
| `tabs` | Access tab URLs to apply correct settings |
| `storage` | Save your preferences locally |
| `alarms` | Schedule page refreshes |
| `activeTab` | Interact with the current tab |
| `scripting` | Run content scripts for element picker |
| `<all_urls>` | Work on any website you choose |

## Your Rights

You can:
- Clear all stored data by removing the extension
- View stored data via Chrome's developer tools
- Modify settings at any time through the extension popup

## Changes to This Policy

We may update this privacy policy occasionally. Changes will be reflected in the "Last Updated" date above.
